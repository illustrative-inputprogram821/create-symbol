## 🎨 How to prepare your SVG files

You will get the best results when your SVG files are clean and simple.

Before you convert a file:
- Use one icon per SVG file
- Make sure the SVG uses closed shapes when possible
- Remove extra layers you do not need
- Keep the view box set correctly
- Use simple fills and strokes
- Save the file in a common SVG format

Good source files help create cleaner SF Symbols.

## 🧩 File tips

For better conversion results:
- Use clear file names
- Avoid very long names
- Keep related icons in one folder
- Do not mix rough drafts with final artwork
- Use a fixed size grid when you create the SVGs

A clean folder structure makes it easier to find the right asset later.

## 🔍 Troubleshooting

### The app does not open
- Try running it again
- Right-click the file and choose Run as administrator
- Move the app to a simple folder like Desktop
- Make sure Windows has not blocked the file

### The SVG does not convert
- Open the SVG in another editor and save it again
- Remove unsupported elements
- Try a simpler icon
- Check that the file ends in `.svg`

### The output looks wrong
- Fix the source SVG first
- Check the view box
- Remove hidden objects
- Use cleaner paths and fewer effects

### The export folder is empty
- Make sure the conversion finished
- Check that you picked a valid output folder
- Try the process again with one file
